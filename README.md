## Board Infinity Assignment
- Simple Web Application using Html, CSS & JavaScript.
- My Board Infinity Assignments.

Board Infinty Assignments 

- Assignment 1
  - Basic Calculator
  - Tip Calculator
  - Stopwatch
- Assignment 2
  - Color Change
  - Random Card Generator
  - Recipe App
